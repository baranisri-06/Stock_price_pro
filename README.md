# Stock_price_pro
# 📈 Tesla Stock Price Prediction

## 📌 Project Overview

This project focuses on predicting Tesla stock prices using Machine Learning. It uses historical stock data to analyze trends and forecast future prices based on past patterns.

---

## 🎯 Objective

To build a simple and effective model that predicts stock closing prices using historical data and visualize the results.

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

---

## 📊 Dataset

The dataset used in this project is **Tesla stock data (tesla.csv)**, which includes:

* Date
* Open
* High
* Low
* Close
* Volume

---

## ⚙️ Methodology

1. **Data Collection** – Load Tesla dataset from CSV file
2. **Data Preprocessing** – Handle missing values and format date
3. **Feature Selection** – Use closing price for prediction
4. **Data Scaling** – Normalize data using MinMaxScaler
5. **Model Training** – Train using Linear Regression
6. **Prediction** – Predict stock prices
7. **Evaluation** – Compare actual vs predicted values using graphs and MSE

---

## 📈 Output

* Stock price visualization
* Moving average trend (optional improvement)
* Actual vs Predicted price graph
* Model evaluation using Mean Squared Error (MSE) and R² score

---

## 🚀 Features

* Time-series data handling
* Trend analysis using moving average
* Next-day price prediction
* Simple and beginner-friendly implementation

---

## ⚠️ Limitations

* Not 100% accurate
* Market influenced by external factors
* Uses only historical data

---

## 🔮 Future Enhancements

* Implement LSTM (Deep Learning)
* Add real-time data
* Include news sentiment analysis

---

## 📌 Conclusion

This project demonstrates how Machine Learning can be used to analyze stock data and predict future prices. It helps understand trends and provides a basic forecasting app
